# Bot-tc-Atualizado-2.0

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/duarttdan/Bot-tc-Atualizado-2.0.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bot-tc-Atualizado-2.0
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage Guide
To start the bot, run:
```bash
node index.js
```
